Strings

A "String" is a type of variable in Python that holds alphanumeric data.

Strings are created in Python by surrounding text with single or double quotes.


                        >>> 'spam and eggs'
                        'spam and eggs'
                        >>> 'doesn\'t'  # use \' to escape the single quote...
                        "doesn't"
                        >>> "doesn't"  # ...or use double quotes instead
                        "doesn't"
                        >>> '"Yes," he said.'
                        '"Yes," he said.'
                        >>> "\"Yes,\" he said."
                        '"Yes," he said.'

Strings

Strings can be stored in variables just like numbers.


                        >>> my_string_variable = 'hi I am a string ðŸ˜›'
                        >>> my_string_variable
                        'hi i am a string ðŸ˜›'

Yep! Strings can contain emoji! (in Python 3)

Strings

Strings can span more than one line.

The '\n' character means "new line".


                        >>> two_line_string = 'This is line 1.\nThis is line 2.'
                        >>> two_line_string
                        'This is line 1.\nThis is line 2.'
                        >>> print(two_line_string)
                        This is line 1.
                        This is line 2.

Strings

Another way to create multi-line strings is with triple-quotes.

Triple-quotes let you leave out the '\n' character.


                        >>> usage = """\
                        ... Usage: thingy [OPTIONS]
                        ...      -h                        Display this usage message
                        ...      -H hostname               Hostname to connect to
                        ... """
                        >>> print(usage)
                        Usage: thingy [OPTIONS]
                            -h                        Display this usage message
                            -H hostname               Hostname to connect to

Strings

Strings can be glued together with the + operator.


                        >>> prefix = 'super'
                        >>> suffix = 'duper'
                        >>> prefix + suffix
                        'superduper'
                        >>> one = '1'
                        >>> two = '2'
                        >>> one + two
                        '12'

Strings

Strings containing numbers can be converted to numbers.


                        >>> int('10')
                        10
                        >>> float('10.9')
                        10.9
                        >>> int('1') + int('2')
                        3
                        >>> int('1.1')

Types

Every value in Python has a "type".


                        >>> type('abc')
                        <class 'str'>
                        >>> type(123)
                        <class 'int'>
                        >>> type(0.1)
                        <class 'float'>

Types

Different types can implement operations in different ways.


                        >>> hundred_str = '100'
                        >>> hundred_str + hundred_str
                        '100100'
                        >>> hundred_int = int(hundred_str)
                        >>> hundred_int + hundred_int
                        200

Types

Not all operations make sense for all types.


                        >>> hundred_str = '100'
                        >>> hundred_int = int(hundred_str)
                        >>> len(hundred_str)
                        3
                        >>> len(hundred_int)
                        Traceback (most recent call last):
                          File "<stdin>", line 1, in <module>
                        TypeError: object of type 'int' has no len()

String Indexing

We can use square brackets ([]) to access an individual character from a string.


                        >>> word = 'Python'
                        >>> word[0]  # character in position 0
                        'P'
                        >>> word[5]  # character in position 5
                        'n'

This is called indexing. The index of the first element is always 0.

String Indexing

Indexes can be negative! In this case they start from the end of the string.


                        >>> word = 'Python'
                        >>> word[-1]  # last character
                        'n'
                        >>> word[-2]  # second-last character
                        'o'
                        >>> word[-6]
                        'P'

String Slicing

The [] notation can also be used to get a substring, or slice of a string.

[i:j:step] where characters starting at position i are included, up to position j (which is excluded).

The step defaults to 1.


                        >>> word = 'Python'
                        >>> word[0:2]  # 0 (included) to 2 (excluded)
                        'Py'
                        >>> word[2:5]  # 2 (included) to 5 (excluded)
                        'tho'

String Slicing

i, j, or both may be omitted when creating a slice.


                        >>> word = 'Python'
                        >>> word[:2]
                        'Py'
                        >>> word[2:]
                        'thon'
                        >>> word[:]
                        'Python'
                        >>> word[:2] + word[2:]
                        'Python'

Lists

A list is a data type that can contain multiple values.

Lists are created with comma-separated values in square brackets ([]).


                        >>> squares = [1, 4, 9, 16, 25]
                        >>> squares
                        [1, 4, 9, 16, 25]

The elements of a list don't have to be all the same type, but usually they are.

Lists

Lists can be indexed and sliced (just like strings).


                        >>> squares = [1, 4, 9, 16, 25]
                        >>> squares[0]
                        1
                        >>> squares[-1]
                        25
                        >>> squares[-3:]  # slicing returns a new list
                        [9, 16, 25]
                        >>> word = 'Python'
                        >>> word[0]
                        'P'

This is no accident! Different types supporting the same operations is a fundamental part of Python.

Lists

List slices can have a step other than 1.


                        >>> squares = [1, 4, 9, 16, 25, 36, 49]
                        >>> squares[0::2]
                        [1, 9, 25, 49]
                        >>> squares[4::-1]
                        [25, 16, 9, 4, 1]

Lists

The len function works on lists just like strings.


                        >>> squares = [1, 4, 9, 16, 25]
                        >>> len(squares)
                        5
                        >>> word = 'Python'
                        >>> len(word)
                        6

Lists

Lists can be concatenated with the + operator, or appended to with their .append method.


                        >>> squares = [1, 4, 9, 16, 25]
                        >>> squares + [36, 49]   # concatenation returns a new list
                        [1, 4, 9, 16, 25, 36, 49]
                        >>> squares.append(36)  # `append` modifies the list in-place
                        >>> squares.append(49)
                        >>> squares
                        [1, 4, 9, 16, 25, 36, 49]

Lists

Unlike strings, lists are mutable, meaning it is possible to change their content.


                        >>> cubes = [1, 8, 27, 65, 125]  # whoops!
                        >>> cubes[3] = 4 ** 3
                        >>> cubes
                        [1, 8, 27, 64, 125]
                        >>> word = 'Pythom'
                        >>> word[5] = 'n'
                        Traceback (most recent call last):
                          File "<stdin>", line 1, in <module>
                        TypeError: 'str' object does not support item assignment

Lists

It is possible to assign to list slices, which can change the size of a list.


                        >>> letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
                        >>> letters
                        ['a', 'b', 'c', 'd', 'e', 'f', 'g']
                        >>> # replace some values
                        >>> letters[2:5] = ['C', 'D', 'E']
                        >>> letters
                        ['a', 'b', 'C', 'D', 'E', 'f', 'g']
                        >>> # now remove them
                        >>> letters[2:5] = []
                        >>> letters
                        ['a', 'b', 'f', 'g']

Lists

Lists can contain other lists.

These are called "nested lists".


                        >>> letters = ['a', 'b', 'c']
                        >>> numbers = [1, 2, 3]
                        >>> nested_list = [letters, numbers]
                        >>> nested_list
                        [['a', 'b', 'c'], [1, 2, 3]]
                        >>> nested_list[0]
                        ['a', 'b', 'c']
                        >>> nested_list[0][1]
                        'b'

The if Statement

The simplest "control flow" statement is the if statement.

It makes the program do something only if the value being tested is true.

The "something" to be run must be indented.


                        >>> value = 3
                        >>> if value > 1:
                        ...     print("The value is greater than 1.")
                        ...
                        The value is greater than 1.
                        >>> if value < 1:
                        ...     print("The value is less than 1.")  # will not run
                        ...
                        >>>

Boolean Values

The if keyword is followed by a boolean expression.

That is, something that is evaluated to be either True or False.

In python, every object can be evaluated in a boolean context.

The following evaluate to false: False, None, 0, "" (the empty string), and empty containers (lists, dictionaries, etc).

Everything else evaluates to true.

Comparison Operators

Comparison operators yield a value of True or False.

< <= > >= == !=


                        >>> 1 > 2
                        False
                        >>> 1.1 <= 2.0
                        True
                        >>> 1 == 2
                        False

Comparison Operators

Comparison operators yield a value of True or False.

is [not]


                        >>> a = [1, 2, 3]
                        >>> b = [1, 2, 3]
                        >>> a == b
                        True
                        >>> a is b
                        False
                        >>> a = b
                        >>> a is b
                        True

Comparison Operators

Comparison operators yield a value of True or False.

[not] in


                        >>> a = [1, 2, 3]
                        >>> 2 in a
                        True
                        >>> 1 not in a
                        False
                        >>> "Hello" in "Hello, world!"
                        True

Combining Comparison Operators

Use the and and or keywords to combine multiple comparisons into a single test.


                        >>> a = 50
                        >>> a >= 50 and a < 60
                        True
                        >>> a < 0 or a > 10
                        True
                        >>> a < 0 or a > 100
                        False

The if Statement

if statements can contain multiple elif parts and one else part.


                        >>> x = int(input("Please enter an integer: "))
                        Please enter an integer: 42
                        >>> if x < 0:
                        ...     x = 0
                        ...     print('Negative changed to zero')
                        ... elif x == 0:
                        ...     print('Zero')
                        ... elif x == 1:
                        ...     print('Single')
                        ... else:
                        ...     print('More than one')
                        ...
                        More than one
